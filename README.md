# DecisionIQ - Smart Comparison Tool
---
## Project Overview

DecisionIQ is a smart comparison tool that helps users make decisions by comparing multiple options based on different criteria.  
It normalizes scores, applies weights, and calculates the total score to determine the best option.  

This tool is useful for students, teams, or anyone who wants to make decisions objectively.

---

## Features

- Add multiple options to compare  
- Add criteria with weights and type (Benefit / Cost)  
- Enter scores for each option based on the criteria  
- Automatically normalize scores  
- Calculate total scores and show the best option  
- Interactive bar chart visualization of results  
- Responsive and clean user interface using Tailwind CSS  

---

## Technology Used

- HTML5  
- CSS3 + Tailwind CSS  
- JavaScript  
- Chart.js for bar charts  

---

## How to Use

1. Add options: Type the name of an option and click **Add**  
2. Add criteria: Enter the criteria name, weight, type, and click **Add**  
3. Enter scores: Fill in the score for each option based on the criteria  
4. Calculate result: Click **Calculate Result** to see the best option and chart  

> The tool normalizes scores automatically and calculates total points.

---

## Notes

- Make sure to add options and criteria before entering scores  
- Recommended browsers: Chrome, Edge, Firefox
